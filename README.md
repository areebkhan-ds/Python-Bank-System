# 🏦 Python Bank Management System

A simple yet powerful **Bank Management System** built using Python and Streamlit.  
This project simulates real-world banking operations like account creation, transactions, and secure authentication.

---

## 🚀 Features

- ✅ Account Creation  
- 🔐 Secure Login (PIN Hashing)  
- 💰 Deposit Money  
- 💸 Withdraw Money  
- 🔁 Transfer Money Between Accounts  
- 📜 Transaction History  
- 💾 Persistent Data Storage (JSON)  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- JSON (for database)  
- OOP (Object-Oriented Programming)  

---

## 📂 Project Structure
bank_app/
│
├── bank.py # Backend logic
├── app.py # Streamlit UI

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
git clone https://github.com/areebkhan-ds/Python-Bank-System.git

cd Python-Bank-System

### 2️⃣ Create virtual environment (optional)

python -m venv venv
venv\Scripts\activate

### 3️⃣ Install dependencies

pip install streamlit

### 4️⃣ Run the app

streamlit run app.py
